# Maison Lignum — site vitrine

Site vitrine pour une entreprise de menuiserie sur mesure haut de gamme (Île-de-France).

## Contenu
- `index.html` — le site complet (une seule page, autonome)
- `images/` — photos des réalisations et visuels

## Mise en ligne (GitHub Pages)
1. Créer un dépôt sur GitHub et y pousser ces fichiers.
2. Dans **Settings → Pages**, choisir la branche `main` et le dossier `/ (root)`.
3. Le site sera publié à l'adresse `https://<utilisateur>.github.io/<dépôt>/`.

Aucune dépendance, aucune compilation : il suffit d'ouvrir `index.html`.
